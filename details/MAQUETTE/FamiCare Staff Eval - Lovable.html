L'app FamiCare est prête. Voici comment démarrer :

  1. Allez sur /setup depuis la page de connexion pour créer le premier compte admin
  2. Connectez-vous en admin → onglet Créer / gérer tests pour ajouter vos tests (Ménage/Repassage et Garde d'enfants) avec leurs questions et options
  3. Onglet Gestion utilisateurs pour ajouter intervenantes & responsables

Fonctionnalités livrées (selon votre structure) :

  • 🔐 Login unique avec redirection par rôle
  • 👩‍🍳 Espace Intervenante : tests disponibles, passage question par question avec image, score immédiat + mention, historique
  • ⚙️ Espace Admin : KPIs + graphiques (Recharts), CRUD tests/questions/options, gestion utilisateurs avec rôles, export CSV
  • 📊 Espace Responsable : vue d'ensemble des scores, fiche détaillée par intervenante

Design inspiré directement du site FamiCare (palette crème/peach/sage, typographie Playfair Display, cards arrondies).
